# e-plantShopping

**Project Name:** e-plantShopping

**Overview:**  
e-plantShopping is a React-based web application for buying and managing plants online. Users can browse various categories of plants, add them to their shopping cart, and see the total quantity and cost of their selected items. The application also integrates Redux for state management to handle cart operations efficiently.

**Features:**
- Browse plants by category
- Add/remove plants to/from the shopping cart
- Track total quantity and total cost dynamically
- Responsive and user-friendly interface
